# CRM API Documentation

API RESTful for Pancake CRM

**Version**: 1.0  
**Base URL**: `https://crm.pancake.vn/api`  
**Auth**: Bearer JWT

## Key Features

- Lead management (CRUD)
- Sources management (Customer Sources)

## Main Endpoints

- **Leads** → `/leads` management
- **Sources** → `/sources` Manage lead sources
